# Week 2 - Arrays

---

## Assignments:
* Scrabble 
  - Every alphabetical character have a score and two players will try their luck with a word where the score is calculated and whoever scores more wins, if score is equal for both players, it's a tie, implemented in C.
  - [Problem-Statement](https://cs50.harvard.edu/x/psets/2/scrabble/)
* Readability 
  - Determines the reading level required to comprehend some text (sentences/paragraphs). Uses Coleman-Liau index to find the reading level, implemented in C.
  - [Problem-Statement](https://cs50.harvard.edu/x/psets/2/readability/)
* Caesar 
  - Convert a plain text to a cipher text based on a numerical key k where k is a positive integer less than $`2^{31}`$ − 26, implemented in C.
  - [Problem-Statement](https://cs50.harvard.edu/x/psets/2/caesar/)
* Substitution 
  - Encrypt messages using a substitution cipher which is a key string of length 26 with unique alphabetical characters in any order, implemented in C.
  -[Problem-Statement](https://cs50.harvard.edu/x/psets/2/substitution/)
