# Project: Ping-Pong Game

## Description:
A simple 2D ping pong game built using Scratch. There are two players that compete against each other towards a target score and the first one who reaches that wins !!.

---

## Features:
* Simple and engaging gameplay
* Dual Player Mode
* Ball speed increase with each edge hit
* Music and sound effects
* Play Online
